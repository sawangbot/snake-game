# 🐍 Snake Game

A classic Snake game with a modern neon aesthetic — built with pure HTML, CSS, and JavaScript.

## 🎮 How to Play

- **Arrow keys** or **WASD** to control the snake
- Eat the red food to grow and score points
- Don't hit yourself!
- The snake wraps around the walls

## ✨ Features

- Dark neon theme with glowing effects
- Score tracking with persistent high score (localStorage)
- Speed increases every 5 food eaten
- Responsive design — works on desktop & mobile
- Touch swipe controls & on-screen D-pad for mobile
- Smooth animations with pixel-perfect rendering

## 🚀 Play

Open `index.html` in any browser. No build step needed.
